Remarks

The Validator class ensures that latitude and longitude values are within their valid ranges:

  • Latitude must be between -90 and 90 degrees.
  • Longitude must be between -180 and 180 degrees.

If a value is out of range or not finite, an exception is thrown via PolylineAlgorithm.Internal.Diagnostics.ExceptionGuard.ThrowCoordinateValueOutOfRange(System.Double,System.Double,System.Double,System.String).
