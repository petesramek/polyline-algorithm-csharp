Remarks

This struct is designed to be immutable and lightweight, providing a simple way to represent geographic coordinates in degrees. It includes validation for latitude and longitude ranges and provides methods for equality comparison and string representation.

Note: The value (0, 0) is a valid coordinate (Gulf of Guinea), but is also treated as the "default" value by PolylineAlgorithm.Coordinate.IsDefault.
